## Description
This CRUD implementation works on TODO Application

Todos can be Created, Read(Display all Todos), Updated(Edited) and Deleted 

## Prerequisites

*npm create vite@latest*
*npm run dev*

## Live Application URL

https://crudtodobyraj.netlify.app/
